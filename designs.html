Next steps — getting the illustrations done

  1. Pick 3–4 favourite concepts from above (catchphrase + tee color combos).
  2. Commission a caricature illustrator. Best options:
    • Fiverr — search "cartoon t-shirt illustration". $50–$200 per design, 3–7 day turnaround.
    • Upwork — higher-end illustrators, $200–$600, better for consistent style across a collection.
    • 99designs — run a contest, get multiple concepts.
    • Local Jamaican illustrators — worth finding one authentic to the culture (Instagram hashtags: #jamaicanart #caribbeanillustrator).
  3. Brief to give the illustrator: send them this page + 3–4 of Oliver's reference photos. Specify: vector style, bold outlines, vibrant Jamaican colors, consistent character across all pieces, transparent PNG + SVG deliverables.
  4. Alternative (faster, cheaper, lower quality): use Midjourney / DALL-E to generate caricatures (~$10/month subscription). Results are inconsistent character-to-character — OK for a single shirt, risky for a collection.
  5. Once illustrations are in, I'll drop them into these exact layouts and we'll have production-ready mockups in an afternoon.
